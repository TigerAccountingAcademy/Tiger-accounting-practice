    About TIGER Accounting Academy

    TIGER Accounting Academy is an interactive learning tool built to help accounting students practise by doing. The goal is simple: make financial statement practice more active, less intimidating, and easier to repeat.

    Current modules include balance sheet practice, income statement practice, cash flow statement practice, journal entries, ratio analysis, exam mode and advanced accounting concept checks.

    Professional angle: This project demonstrates initiative, accounting interest, problem-solving and willingness to build practical tools for student learning.
